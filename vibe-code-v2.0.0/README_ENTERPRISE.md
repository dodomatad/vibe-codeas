## 🚀 Quick Start

\`\`\`bash
# Install dependencies
pnpm install

# Setup environment
cp .env.example .env.local
# Edit .env.local with your API keys

# Run tests
pnpm test:coverage

# Start development server
pnpm dev

# Build for production
pnpm build
pnpm start
\`\`\`
